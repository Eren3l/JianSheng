# 需求规格说明

## 项目名称
历史粘贴板 (Clipboard History)

## 用户画像
Windows 用户，日常办公频繁使用复制粘贴，需要回溯近期复制过的内容和图片。

## 核心功能

### 1. 剪贴板监听
- 打开软件后自动监听系统剪贴板变化
- 支持文字内容（纯文本）
- 支持图片内容（截图、复制的图片）
- 关闭软件即停止监听

### 2. 内容查看
- 以时间倒序（最新在前）排列展示
- 列表形式展示，每条显示时间和内容摘要
- 点击某条可展开查看完整内容或图片
- 置顶内容始终在最上方显示

### 3. 搜索与筛选
- 输入关键词搜索文字内容
- 可按日期范围筛选（今天、近3天、近7天、自定义）
- 搜索结果高亮匹配文字

### 4. 内容管理
- 置顶：重要内容可置顶，置顶内容不过期
- 删除：可单独删除某条记录（软删除）
- 批量操作：暂不实现，保持简洁

### 5. 自动清理
- 用户可设置保留天数：1天 / 3天 / 5天
- 超过保留天数的内容自动标记为过期
- 过期时弹出提示，用户确认后再清理
- 置顶内容永不过期

### 6. UI 要求
- 简洁直观，无需学习成本
- 主色调：淡蓝色 (#5B9BD5)
- 窗口固定大小约 400×600 像素
- 支持右键菜单操作

## 非功能需求
- 启动速度：1 秒内显示窗口
- 图片存储：压缩至 100KB 以内
- 数据库：单文件 SQLite，用户无感知
- 兼容性：Windows 10/11 + Python 3.8+

## 不做的功能（明确边界）
- 不支持富文本（保持简单）
- 不支持网络同步
- 不支持快捷键自定义
- 不支持多选批量操作（保持界面简洁）
- 不支持云备份

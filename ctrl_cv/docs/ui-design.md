# UI 设计规范

## 设计原则
- **简洁**：减少视觉噪音，让内容成为焦点
- **直观**：操作符合 Windows 用户习惯，无需学习
- **轻量**：窗口小巧（400×600），不占屏幕空间

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | #F0F6FB | 淡蓝白底色，整个窗口背景 |
| 卡片背景 | #FFFFFF | 白色卡片，与背景区分 |
| 置顶卡片 | #E3F0FA | 浅蓝底色，突出显示 |
| 主色调 | #5B9BD5 | 按钮背景、选中状态 |
| 主色深 | #3A7BC8 | 按钮悬停、标题栏、边框 |
| 文字主色 | #333333 | 正文、标题 |
| 文字辅助 | #888888 | 时间戳、提示文字 |
| 危险色 | #E74C3C | 删除按钮 |
| 边框色 | #DDE8F0 | 卡片分隔线 |

```
┌──────────────────────────────────────┐
│  历史粘贴板                    ─ □ × │  ← 标题栏（主色深）
├──────────────────────────────────────┤
│  [🔍 搜索文字...    ]  [日期▼] [⚙]   │  ← 工具栏（高 40px）
├──────────────────────────────────────┤
│                                      │
│  ┌────────────────────────────────┐  │
│  │ 📌 复制的文字内容预览... 12:30 │  │  ← 置顶卡片
│  ├────────────────────────────────┤  │
│  │ 另一段文字内容预览...   12:15  │  │  ← 普通卡片
│  ├────────────────────────────────┤  │

│  ├────────────────────────────────┤  │
```

窗口尺寸：420 × 600 像素（宽×高），可适当缩放

## 组件规格

### 搜索栏
- 搜索框：左侧带 🔍 图标，placeholder "搜索文字..."
- 日期下拉："全部时间" / "今天" / "近3天" / "近7天"
- 设置按钮：齿轮图标 ⚙，最小化设计

### 列表卡片
- 每张卡片高度自适应（单行约 40px，展开后可达 200px）
- 左侧显示文字摘要（图片则显示"🖼 图片"）
- 右侧显示时间（HH:MM 或 MM-DD HH:MM 格式）
- 置顶卡片带 📌 图标，淡蓝底色
- 点击展开显示完整内容或图片预览

### 右键菜单
- 置顶 / 取消置顶
- 复制（将内容重新放入剪贴板）
- 删除

### 详情面板（点击展开）
- 文字：完整文字内容，可选择复制
- 图片：显示压缩后的图片，有"复制图片"按钮

### 设置弹窗
- 保留天数单选：1天 / 3天 / 5天
- 立即清理按钮
- 关闭按钮

## 交互规范
- 左键点击 → 展开/收起详情
- 右键点击 → 弹出操作菜单
- 搜索框实时过滤，输入即搜
- 日期筛选立即生效
- 删除时有确认对话框
